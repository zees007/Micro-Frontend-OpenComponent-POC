How to run the project in local:

1. Import the project in your IDE
2. Install OC first by executing below CLI command:

   npm install oc -g

3. Now deploy all the components on local registry by running below command:
   
   oc dev . 3030

4. Now open http://localhost:3030 to see the repo of all components.

5. To view the entire application look and feel combining all components just hit below url:

    http://localhost:63342/oc_new/host/index.html?_ijt=147fhonvsmv7i0fuchhh71hae6
